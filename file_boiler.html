File Path & Boilerplate

File Path in HTML

HTML files can link images, CSS, or JS files using relative or absolute paths.

<img src="images/photo.jpg">   <!-- relative path -->
<img src="https://example.com/photo.jpg">  <!-- absolute path -->

HTML Boilerplate Code

Basic structure of any HTML page.

<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  Content goes here
</body>
</html>

DIV + Class + Id

DIV is used to create sections. You can assign class or id for styling or JS.

<div id="header">Header</div>
<div class="container">Content</div>
