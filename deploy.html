Deploy & Media

Media Tag

HTML supports images, audio, video tags:

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  </audio>
  

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  </video>

Multi Page Website

Create multiple HTML files and link using <a> tag for navigation.

<a href="page2.html">Go to Page 2</a>

Deploy First Project

Steps to deploy:

  1. Save your HTML files.
  2. Upload to hosting (Netlify, GitHub Pages, Vercel).
  3. Access online with your domain.
