HTML Basics

HTML Headings

HTML provides six levels of headings: <h1> to <h6>. Headings are used to define titles and sub-titles.

<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Smaller Heading</h3>

Paragraph

Paragraphs are defined using the <p> tag. They separate text into readable blocks.

<p>This is a paragraph.</p>

Anchor Tag

Anchor (<a>) tag is used to create hyperlinks to other pages or websites.

<a href="https://www.google.com">Go to Google</a>

Image Element

Images are inserted using <img> tag. The src attribute specifies the image path, and alt provides alternative text.

<img src="mypic.jpg" alt="Description">
