Internet Working

How Internet Works

The Internet is a global system of interconnected computer networks. It works using protocols like TCP/IP to allow communication between devices worldwide.

Key Components

  • Client: The user’s device (laptop, phone).
  • Server: The machine hosting websites or data.
  • ISP: Internet Service Provider, provides internet access.
  • DNS: Resolves domain names to IP addresses.

Workflow

Steps of how data travels on the internet:

  1. User enters a URL in the browser.
  2. DNS translates domain to IP.
  3. Request is sent to the server.
  4. Server responds with HTML data.
  5. Browser renders the page.

Example

When you type https://www.google.com:

Browser → DNS → Google Server → Response → Browser shows page
