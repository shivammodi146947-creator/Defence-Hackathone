HTML Lists & Tables

Ordered List

Ordered lists use <ol> tag. Items are numbered automatically. 

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

Unordered List

Unordered lists use <ul> tag. Items are shown with bullets. 

<ul>
  <li>Item A</li>
  <li>Item B</li>
</ul>

Nested Lists

You can create lists inside lists (nested lists).

<ul>
  <li>Item 1
    <ol>
      <li>Subitem 1</li>
      <li>Subitem 2</li>
    </ol>
  </li>
</ul>

Tables

Tables are defined with <table>. Rows use <tr>, cells use <td> or <th>.

Normal Table

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>30</td>
    <td>London</td>
  </tr>
</table>

Nested Table

Tables can be nested inside a cell of another table.

<table border="1">
  <tr>
    <th>Team</th>
    <th>Players</th>
  </tr>
  <tr>
    <td>Team A</td>
    <td>
      <table border="1">
        <tr><td>Alice</td></tr>
        <tr><td>Bob</td></tr>
      </table>
    </td>
  </tr>
  <tr>
    <td>Team B</td>
    <td>
      <table border="1">
        <tr><td>Charlie</td></tr>
        <tr><td>David</td></tr>
      </table>
    </td>
  </tr>
</table>
